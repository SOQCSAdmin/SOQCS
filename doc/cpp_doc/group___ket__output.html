Prints a ket of the list given its position in the list.

Parameters
intiket Index of ket to be printed.
intformat Format of the output.
0 Straightforward form:
The occupation values are printed in ascending order. The method reverts to this format independently of the configuration if the pointer to the circuit "qocircuit* qoc" is Null
Example:
| 0, 0, 2, 0 >

1 Human readable format: | [occupation]H/V(wavefunction)channel >
Detailed format:
{{ | {{[o1]}o1>1}{H/V}{(t1)}ch1 > } o1>0},...,{{ | {{[on]}o1>1}{{H/V}{(tn)}chn > } on>0}
Example:
| [2]V(0)0 >

Legend:
chi:Channel i, mi: polarization i, ti: wavefunction i.
{} Optional values only to be printed if there is more than one possibility.
{{}v>i} Optional values only to be printed if the variable v>i.
qocircuit*qoc Circuit to which the ket is related.
