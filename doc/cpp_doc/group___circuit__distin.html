Detailed Description

List of advanced circuit elements to configure the photon emission model and the distinguishability between photons in the circuit.

veci qocircuit::emitter ( int  npack,
matd  packets 
)

Adds an emitter to the circuit using a matrix with the parameters needed to create a packet model of the emitted photons.

Parameters
intnpack Number of packets.
matdpackets Matrix that contains the definition of the packets. Each columns contains the information of a single packet with its parameters in rows:
  • 1st row: Packet numbers.
  • 2nd row: Packet central times.
  • 3rd row: Packet central frequencies.
  • 4th row: Packet width or decay time.
Returns
Vector with the new packet numbers indexed by the old ones ( Note that the packet numeration changes on emission if detectors are configured to clock mode 1).

◆ emitter() [4/4]

void qocircuit::emitter ( matc  D)

Adds an emitter to the circuit using a matrix of overlap coefficients between packets.

Parameters
matcD Overlap coefficient matrix for the wavepackets of the initial states.
KnowIss:
When two identical fully overlapping packets are present the library does not provide a correct result. A workaround is to define the overlap as 0.999999 with as many 9's as possible. This bug is inherited from the mathematical library Eigen 3.
