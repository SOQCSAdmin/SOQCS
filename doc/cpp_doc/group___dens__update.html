◆ add_reduced_state()

void dmatrix::add_reduced_state ( int  ndec,
mati  def,
veci  chlist,
state *  input,
qocircuit *  qoc 
)

Adds a new state only if the detector conditions in def are fulfilled. The resulting states are then reduced by tracing out the channels in chlist.
Intended for internal use of the library .

Parameters
intndec Number of detectors. It has to be the number of columns in def.
matidef Definition of the conditional detection. "def" is a 2xn matrix where in the first row are defined the channels where the conditional detection takes place while in the second row is the number of photons to be detected.
vecichlist. List of channels to be traced out from the states added to the density matrix.
state*input State to be added to the density matrix.
qocircuit*qoc Circuit to which the state is related.

◆ calc_measure() [1/2]

dmatrix * dmatrix::calc_measure ( qocircuit *  qoc)

Calculate measure. It means to remove degrees of freedom depending on the circuit configuration of the detectors. ( Circuit version ).

Parameters
qocircuit*qoc Circuit to which the density matrix is related.
Returns
A new smaller matrix with the removed degrees of freedom.
KnowIss:
Non-general assumptions about the measurement process are taken if the density matrix information is reduced (in agreement to the configuration of the detectors).
