Quantum dot state generator. It creates a state compatible with the photon emission of the quantum dot specified by the next parameters. The parameter "ch" provides information about the configuration of the emitted photons.

Parameters
matich Photon configuration matrix.
For each emitted photon we must define (in column) three numbers. Channel and wave function numbers in case photons are horizontally H or vertically V polarized. There are as many columns as photons defined. Photons are created in pairs using the QDPairs function therefore "ch" has to be defined with an even number of columns. Distinguishability is defined in the configuration of the emitter circuit element.
doublek Fraction of the emitted photon pairs which originate both from a XX-X cascade in presence of background light or multiphoton emission.
doubleS S/hbar Fine Structure Splitting (FSS) constant.
doubletss Characteristic time of spin-scattering tss/tx.
doublethv Characteristic time of cross-dephasing thv/tx.
qocircuit*qoc Circuit to which the state to be created is related.

Quantum dot photon pair generator. It creates a photon pair as it would be generated by a quantum dot excited by a laser pulse. Quantum dots emit Bell states, correlated states and random states with a certain probability. Therefore "QDPair" calls the functions "Bell", "Corr" , "Rand".

Parameters
inti_ch0 Channel were the first photon is created.
inti_ch1 Channel were the second photon is created.
vecii_t; Vector with the wavepacket numbers to be assigned to each photon.
doublek Fraction of the emitted photon pairs which originate both from a XX-X cascade in presence of background light or multiphoton emission.
doubleS S/hbar Fine Structure Splitting (FSS) constant.
doubletss Characteristic time of spin-scattering tss/tx.
doublethv Characteristic time of cross-dephasing thv/tx.
qocircuit*qoc Circuit to which the state to be created is related.
