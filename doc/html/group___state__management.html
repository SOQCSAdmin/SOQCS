state::state ( int  i_level,
int  i_maxket,
int *  i_vis 
)

Creates a state specifying the maximum number of kets and the vector of equivalence between state levels and circuit defined levels. Intended for internal use. Useful to use after post-selection where some of the levels have been eliminated and the level ordering does not coincide anymore with the one defined by the circuit.

Parameters
inti_level Number of levels to describe the state.
inti_maxket Maximum number of different kets in the summation.
inti_vis Vector that translates the state levels to the circuit levels defined by the circuit indices.
