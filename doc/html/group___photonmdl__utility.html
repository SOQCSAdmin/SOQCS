◆ create_packet_mtx()

matd photon_mdl::create_packet_mtx ( )

Coverts the photon model into a matrix description that many subroutines can use. It is a transformation in the way information is stored. This routine is intended for internal use.

Returns
A photon packet matrix that contains the same information in a different a convenient order to do certain calculations.

◆ return_packet_def()

mati photon_mdl::return_packet_def ( )

Returns the packet definition in a three row format. This is:

  • 1st-Row: Packet number
  • 2nd-Row: Index to the time matrix.
  • 3rd-Row: Index to the frequency matrix.
    Returns
    Three row packet definition.
