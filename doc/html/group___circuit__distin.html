void qocircuit::emitter ( int  npack,
matd  packets,
char  ckind,
int  rand 
)

Adds an emitter to the circuit using a physical photon model of the emitted wavepackets parametrized by the elements in the matrix "packets".
Each columns contains the information of a single packet with its parameters in rows:

  • 1st row: Packet numbers.
  • 2nd row: Packet central times.
  • 3rd row: Packet central frequencies
  • 4th row: Packet width or decay time
Parameters
intnpack Number of packets.
matdpackets Matrix that contains the definition of the packets.
charckind Packet model 'G': Gaussian/ 'E': Exponential
intrand Initial flight phase computation.
0= No compute.
1=Same than flight times.
2=Random times (within a certain distribution).

◆ emitter() [4/4]

void qocircuit::emitter ( matc  D)

Adds an emitter to the circuit using a coupling model. The coupling model are the overlapping coefficients of a non-orthornormal base of spatial wavefunctions provided in the matrix D.

Parameters
matcD Overlapping coefficient matrix for the wavefunctions of the initial states.
KnowIss:
If we have two identical full overlapping packets the library does not provide a correct result. The overlapping in this case has to be 0.999999 with as many 9's as possible. This bug is inherited from Eigen 3.
