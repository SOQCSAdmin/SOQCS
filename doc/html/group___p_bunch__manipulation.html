◆ alternative()

void ph_bunch::alternative ( )

After alternative is called new photons are added to a new state that exists in superpostion with the previous ones. Useful to define entangled sets of photons. Example:

bunch->add_photons(0, 0, circuit);
bunch->add_photons(1, 0, circuit);

means an state |11> while

bunch->add_photons(0, 0, circuit);
bunch->alternative();
bunch->add_photons(1, 0, circuit);

means 1/sqrt(2)(|10>+|11>). A weight/amplitude one is assumed if nothing is specified and the state is normalized when sent to the circuit.
